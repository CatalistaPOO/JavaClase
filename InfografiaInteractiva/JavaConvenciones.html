Convenciones de Nomenclatura en Java

Una guía visual para escribir código limpio, legible y profesional, siguiendo las mejores prácticas.

Clases e Interfaces: `PascalCase`

Los nombres de clases e interfaces deben ser sustantivos que comiencen con una letra mayúscula y sigan la convención PascalCase. Esto distingue claramente las plantillas de los objetos de las variables y los métodos, mejorando la legibilidad del código.

**Ejemplos:**

class Coche { ... }
class DireccionDeEnvio { ... }
interface InterfazComparable { ... }
abstract class GestorDeConexionAbstracto { ... }

Métodos y Variables: `camelCase`

Los métodos (verbos) y las variables (sustantivos) utilizan camelCase. Esta convención es fundamental para la mantenibilidad del código, ya que facilita la identificación de la función de cada elemento.

**Ejemplos de Métodos:**

public void calcularTotal() { ... }
public String obtenerNombreCompleto() { ... }
public void establecerEdad(int edad) { ... }

**Ejemplos de Variables:**
int edad = 30;
String nombreUsuario = "Juan";
boolean estaActivo = true;

La aplicación de camelCase, junto con nombres descriptivos, es un pilar de la claridad del código.

Constantes: `SCREAMING_SNAKE_CASE`

Las constantes, declaradas como `public static final`, utilizan mayúsculas y guiones bajos. Esto las hace destacar visualmente, indicando que su valor es inmutable y está disponible globalmente a través de la clase.

**Ejemplos:**

public static final double TASA_IMPUESTO = 0.21;
public static final int MAXIMO_INTENTOS = 5;
public static final String RUTA_ARCHIVO = "/home/data.txt";

Una constante bien declarada combina visibilidad, estado estático, inmutabilidad y una convención de nomenclatura clara.

Paquetes: `minúsculas`

Los nombres de los paquetes se escriben completamente en minúsculas para evitar conflictos en sistemas de archivos. La convención más común es usar el nombre de dominio de la organización invertido.

**Ejemplos:**

package com.google.android.gms;
package org.apache.commons.lang3;
package com.miempresa.aplicacion.modelo.datos;

Tipos Genéricos

Para los tipos genéricos, se usan letras mayúsculas de una sola letra para mantener el código conciso. La convención ayuda a entender rápidamente el propósito del tipo genérico en su contexto.

**Ejemplos:**

class Caja<T> { T contenido; }
interface List<E> { void agregar(E elemento); }
interface Map<K, V> { ... }

Los tipos genéricos como T, E, K y V son clave para un código conciso y seguro en Java.

Ejemplo de Código Completo

Este ejemplo de código integra todas las convenciones de nomenclatura en un solo programa, incluyendo paquetes, clases, variables, métodos, constantes y genéricos. Puedes copiarlo y ejecutarlo para ver las convenciones en acción.


package com.miempresa.concesionario;

import java.util.ArrayList;
import java.util.List;

// Clase (PascalCase) que representa un vehículo
class Coche {
    // Constante (SCREAMING_SNAKE_CASE)
    public static final String TIPO_COMBUSTIBLE_DEFAULT = "Gasolina";

    // Variables de instancia (camelCase)
    private String marca;
    private String modelo;
    private int anoDeFabricacion;

    // Constructor (PascalCase para la clase)
    public Coche(String marca, String modelo, int anoDeFabricacion) {
        this.marca = marca;
        this.modelo = modelo;
        this.anoDeFabricacion = anoDeFabricacion;
    }

    // Métodos (camelCase)
    public void mostrarInformacion() {
        System.out.println("Coche: " + marca + " " + modelo + " del año " + anoDeFabricacion);
    }

    public String obtenerMarca() {
        return this.marca;
    }
}

// Clase principal que usa las otras clases
public class Concesionario {

    public static void main(String[] args) {
        // Uso de clases y objetos
        Coche miCoche = new Coche("Toyota", "Corolla", 2022);
        miCoche.mostrarInformacion();

        // Uso de constantes
        System.out.println("El tipo de combustible por defecto es: " + Coche.TIPO_COMBUSTIBLE_DEFAULT);

        // Uso de variables y métodos
        String marcaDelCoche = miCoche.obtenerMarca();
        System.out.println("La marca del coche es: " + marcaDelCoche);

        // Uso de tipos genéricos (List)
        List<Coche> inventario = new ArrayList<>();
        inventario.add(miCoche);
        inventario.add(new Coche("Honda", "Civic", 2023));

        System.out.println("\nInventario de coches:");
        for (Coche coche : inventario) {
            coche.mostrarInformacion();
        }
    }
}
